AgendaFlow Belém
Plataforma SaaS de Automação de Agendamentos para Microempreendedores utilizando n8n, IA e WhatsApp
1. Visão Geral do Projeto

O AgendaFlow Belém é uma plataforma SaaS voltada para Microempreendedores Individuais (MEIs), permitindo que seus clientes realizem agendamentos através do WhatsApp enquanto o empreendedor acompanha toda a operação em um painel web.

O diferencial da solução está na combinação de:

WhatsApp como canal principal de atendimento;
Automações utilizando n8n;
Inteligência Artificial para atendimento;
Gestão centralizada de agendamentos;
Reagendamento automático;
Redução de faltas (No-Show);
Baixo custo operacional.
